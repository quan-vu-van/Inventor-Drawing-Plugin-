MCG Create Dummy Detail / Section

Autodesk Inventor 2023 · Drawing Add-In · v1.0.0

Create Dummy Detail/Section is a palette tool for Autodesk Inventor 2023 that lets you place visual section markers and detail boundaries on drawing documents (.idw) without creating real Section or Detail views. The output is clean sketch geometry that exports cleanly to AutoCAD DWG.

Why "dummy"? Real Inventor Section/Detail views generate extra hidden geometry and view objects that bloat exported DWG files. This add-in places lightweight sketch markers that look identical but stay clean on export.

Steps

  1. Close Autodesk Inventor if it is currently running.
  2. Copy all files below into C:\CustomTools\Inventor\ (create the folder if it does not exist):
    • MCG_InventorCreateDummyDetailSection.dll
    • MCG_InventorCreateDummyDetailSection.addin
    • Symbol Sections.idw
    • Install_AutoLoadInventorAddin.bat
    Final layout:
    C:\CustomTools\Inventor\
      ├─ MCG_InventorCreateDummyDetailSection.dll
      ├─ MCG_InventorCreateDummyDetailSection.addin
      ├─ Symbol Sections.idw
      └─ Install_AutoLoadInventorAddin.bat
  3. Run Install_AutoLoadInventorAddin.bat (double-click it inside C:\CustomTools\Inventor\).
    The script scans the folder for *.addin files and copies them into %APPDATA%\Autodesk\Inventor 2023\Addins\ so Inventor will discover the add-in on startup. The DLL and Symbol Sections.idw stay in C:\CustomTools\Inventor\ — the .addin file references them by absolute path. Press any key to close the script window when it shows Success.
  4. Open Autodesk Inventor 2023 and open any Drawing (.idw) file.
  5. Verify the add-in loaded:
    • Look for the MCG TOOLS tab in the ribbon → Drawing panel → Dummy Tools button.
    • Or go to Tools → Add-In Manager and confirm MCG_InventorCreateDummyDetailSection is listed and ticked as Loaded.
    If the button does not appear, see Troubleshooting.
Important: Symbol Sections.idw must remain in C:\CustomTools\Inventor\ alongside the DLL. The add-in opens this file automatically when section symbols are missing from your drawing.
Multiple add-ins: the same script handles any number of MCG add-ins — drop more *.dll + *.addin file pairs into C:\CustomTools\Inventor\ and run the script again. All MCG add-ins will share the same MCG TOOLS ribbon tab.
Uninstall: close Inventor, then delete MCG_InventorCreateDummyDetailSection.dll from C:\CustomTools\Inventor\ and MCG_InventorCreateDummyDetailSection.addin from %APPDATA%\Autodesk\Inventor 2023\Addins\.

Opening the Panel

  1. Open a Drawing (.idw) file in Inventor 2023.
  2. Click the MCG TOOLS tab in the ribbon.
  3. In the Drawing panel, click Dummy Tools.
  4. The Drawing Center palette opens and docks to the right side of the screen.
The Dummy Tools button is only available in the Drawing environment. The MCG TOOLS tab itself appears on Part, Assembly, and Drawing ribbons (shared with other MCG add-ins), but the Dummy Tools button only shows when a .idw drawing is active.

Required Section Symbols

Creating a dummy section requires 4 SketchedSymbol definitions to exist in your drawing. These are the section arrow markers, one for each viewing direction:

Symbol NameUse
SECTION_L-RVertical section line, viewing direction Left → Right
SECTION_R-LVertical section line, viewing direction Right → Left
SECTION_U-DHorizontal section line, viewing direction Up → Down (top-to-bottom)
SECTION_D-UHorizontal section line, viewing direction Down → Up (bottom-to-top)
These symbols are provided in Symbol Sections.idw shipped with the add-in.

Importing Symbols (one-time per drawing)

When you click Create Dummy Section on a drawing that does not yet have the 4 required symbols, the add-in automatically opens Symbol Sections.idw and shows a dialog with import instructions. You then copy the symbols once via Inventor's UI — they stay in the drawing forever after.

Why a one-time manual copy?

Inventor's API for copying SketchedSymbol definitions does not transfer the solid hatch fill used in the arrow heads. Manual copy through Inventor's UI preserves everything intact.

How to copy

  1. Click Create Dummy Section in the palette.
  2. If symbols are missing, the add-in opens Symbol Sections.idw and shows a dialog.
  3. In the opened library file's Browser tree, expand Sketched Symbols.
  4. For each symbol (SECTION_L-R, SECTION_R-L, SECTION_U-D, SECTION_D-U):
    • Right-click the symbol name → Copy.
  5. Switch back to your own drawing (Window menu or click the file tab).
  6. In your drawing's Browser, right-click the Sketched Symbols node → Paste (repeat for each symbol).
  7. Click Create Dummy Section again — the workflow now proceeds.
After copying once, the 4 symbols stay permanently in your drawing. Subsequent dummy sections in this drawing run instantly without prompting.

Create Dummy Section

Places two section arrow symbols connected by a 50 mm sketch stub on each end, simulating a real Section view's section line — without creating a Section view.

Workflow

  1. (Optional) Pre-select a DrawingView on the drawing. If you do, the add-in skips the "Click VIEW" step and uses the selected view directly.
  2. Click Create Dummy Section in the palette.
  3. Click on a VIEW in the drawing (skipped if a view was pre-selected).
  4. Click START point of the section line.
  5. Click END point. The point auto-snaps to horizontal or vertical alignment with the start. A dashed preview line follows the cursor.
  6. Click DIRECTION side — the side from which the section is being viewed.
  7. A popup asks for the section name (e.g., A1). Press OK.
  8. The add-in places the correct directional symbol at both ends and writes the name into the symbol's prompt field.

What is created

  • 2 sketch line stubs (each 50 mm × view scale denominator) at the ends of the section line, dash-dotted.
  • 2 SketchedSymbol instances at the midpoints of the stubs, with the section name written in the middle prompt field.
  • Dimension constraint of 50 mm × scale denominator on each stub.
  • Collinear and Horizontal/Vertical constraints between the two stubs.
Right-click at any time during picking to cancel.

Create Dummy Detail

Places a detail boundary (circle or rectangle) with a label on the drawing, simulating a real Detail view callout — without creating a Detail view.

Workflow

  1. (Optional) Pre-select a DrawingView on the drawing to skip the view-pick step.
  2. Click Create Dummy Detail. A floating dialog appears.
  3. Configure in the dialog:
    • Detail name — e.g., A
    • Boundary shape — Circle or Rectangle
    • Connection line — checked: draws a leader to a separate label position; unchecked: places label directly next to the boundary
    Click OK.
  4. Click on a VIEW (skipped if pre-selected).
  5. For Circle: click CENTER, then edge point (defines radius).
    For Rectangle: click CORNER 1, then CORNER 2 (opposite).
    A live preview of the shape follows the cursor.
  6. Click LABEL position — where the text label should appear. A live connection-line preview follows the cursor (if connection line is enabled).
  7. The add-in creates the boundary, optional connection line + underline, and the label text.

What is created

  • Boundary geometry (circle or rectangle) in dash-dotted style with a dimension constraint (radius for circle, width + height for rectangle).
  • If connection line is enabled: a dash-dotted line from boundary to label position, plus a horizontal underline shelf, plus the label text on top of the shelf.
  • If connection line is disabled: just the label text at the picked position.
  • Label text uses the ISOCPEUR font at 5.25 mm height.

Stub Length Formula

Section stubs scale automatically with the view's scale ratio so they remain readable at any zoom level.

Stub length on paper = 4 × scale denominator (mm)

Troubleshooting

Dummy Tools button is not visible in the MCG TOOLS tab

  • Open Tools → Add-In Manager and check that MCG_InventorCreateDummyDetailSection is listed and ticked as Loaded.
  • Verify all files exist in C:\CustomTools\Inventor\ (DLL + .addin + Symbol Sections.idw), then re-run Install_AutoLoadInventorAddin.bat and restart Inventor.
  • Check that %APPDATA%\Autodesk\Inventor 2023\Addins\MCG_InventorCreateDummyDetailSection.addin exists.
  • The button only appears in the Drawing environment — open a .idw file first.
  • If load failed, check log file at C:\CustomTools\Inventor\logs\MCG_InventorCreateDummyDetailSection.log

Symbol arrows are missing solid fill / appear hollow

This means the symbols were copied via the API fallback (which does not transfer solid hatch). Delete the symbols from your drawing's Browser → Sketched Symbols, then click Create Dummy Section and follow the manual copy procedure described in Importing Symbols.

Symbol Sections.idw not found

Verify Symbol Sections.idw sits in the same folder as the DLL (C:\CustomTools\Inventor\). The add-in looks for it relative to the loaded DLL location.

Status bar shows "Click ngoai view" / picking does not advance

You clicked outside the bounding box of any DrawingView. Click again inside a view to select it. Or: pre-select a view in the drawing (it gets highlighted), then click Create — the view-pick step is skipped.

Click at any time to cancel

Right-click during any picking step to abort. The add-in cleans up and returns to idle.

Text label appears too large or wrong size

The label uses a fixed paper size (5.25 mm for Detail labels). On views with extreme scale ratios the relative size may seem unusual. This is correct behavior — the text is sized for the printed sheet, not the model.
